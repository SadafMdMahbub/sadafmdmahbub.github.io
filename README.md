# 💍 Wedding Invitation Card

### A beautiful, interactive digital wedding invitation with an envelope-opening animation, event slides, countdown timer, and background music.

</div>

---

## ✨ Features

- **✉️ Envelope Opening** – Tap to open a beautifully animated envelope
- **📜 Event Cards** – Swipe through 5 event slides (Intro, Holud, Akt, Wedding, Walima)
- **⏱️ Countdown Timer** – Live countdown to each wedding event
- **🎵 Background Music** – Auto-playing instrumental with a toggle button
- **📱 Mobile-friendly** – Touch swipe support & responsive design
- **⌨️ Keyboard Navigation** – Arrow keys to navigate cards on desktop
